小組期末專案
=======================

- [ ] 原始碼及數據
    - **資料夾** `src/`，內含所有 Rscript、相依檔案 (例如，原始資料) 以及數據 (分析產出之資料)
    - `src/` 內的檔案**有一份 `README.md`**，內容簡要說明每個 Rscript 的功用以及產出之資料的內容
- [ ] 書面報告
    - 檔名：`G<組別>_report` (例: `G01_report.pdf`)
    - 格式：`.pdf`
- [ ] 投影片 (口頭報告組)
    - 檔名：`G<組別>_slides` (例：`G01_slides.pdf`)
    - 格式：`.pdf`
